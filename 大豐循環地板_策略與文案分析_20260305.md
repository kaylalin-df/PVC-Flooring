# Part A：商業策略分析

---

## A1. 處境重述

大豐環保是台灣知名的資源回收與環保服務企業，擁有完整的清運車隊、拆除工班、廢棄物處理許可與超過 10,000 家 B2B 企業客戶。現在要以**獨家經銷**的方式，與製造夥伴長虹合作，推出 PVC 循環再生地板產品。

**核心處境**：
- 大豐在環保產業是知名品牌，但在**建材市場認知度為零**
- 產品具備獨特的「100% 可回收循環再製」賣點，但台灣消費者對「環保地板」概念幾乎無感
- 走獨家經銷而非自製，輕資產模式降低風險，但利潤空間受限於長虹牌價七五折
- 市場時機窗口確實存在：營建廢棄物取締加碼、ESG 法規強制化、碳費開徵
- 台灣 PVC 地板回收率 0%，這是一個零競爭的藍海——但也意味著市場教育成本高
- 目標 4 月上線推動，時間極度緊迫

**要解決的核心矛盾**：如何在一個對「環保」不買單的市場，用環保產品打出差異化，並在建材業零基礎的情況下快速建立通路？

### 市場輪廓

| 競爭者回收能力 | **全部為零** |
| 驅動力 | ESG 法規、綠建材法規、營建廢棄物取締、碳費 |

### 3 個定位角度

| 定位 | 一句話 | 適合客群 | 優勢 | 風險 |
|------|--------|---------|------|------|
| **A. 循環經濟解方** | 「台灣唯一的 PVC 地板循環回收方案」 | 企業ESG、建商 | 獨特、無競爭、故事性強 | 市場教育成本高、「回收」可能聯想低品質 |
| **B. 一條龍省事** | 「拆除→回收→鋪設，一通電話全搞定」 | 物業管理、總務、建商 | 直接解決痛點、易理解 | 差異化不夠鮮明、競爭者可模仿服務模式 |

## A5. 三種策略打法

### 策略一：「通路先行」— 先鋪經銷網，靠經銷商推量

| 項目 | 內容 |
|------|------|
| **核心邏輯** | 簽 25+ 家經銷商，靠經銷商的既有客戶關係推銷，大豐專注品牌+供貨 |
| **優勢** | 快速擴大覆蓋範圍、不需大量業務人力、經銷商自帶客戶 |
| **風險** | 經銷商未必主推（利潤不夠高就放架上不賣）、品牌控制力弱、定價被壓 |
| **適合情境** | 產品定價有明顯競爭力、經銷商有利可圖 |
| **首年營收預估** | NTD 3,000-6,000 萬 |

### 策略二：「案例驅動」— 先做 3-5 個標竿案場，靠口碑擴散

| 項目 | 內容 |
|------|------|
| **核心邏輯** | 鎖定 3-5 個高能見度案場（上市櫃企業總部、科技廠），用優惠價或免費試鋪取得成功案例，再靠案例說話開拓市場 |
| **優勢** | 案例是 B2B 最強的銷售工具、可同步產出 ESG 數據實績、降低客戶決策風險 |
| **風險** | 前期營收低（優惠價或免費）、案場施工品質必須完美、速度較慢 |
| **適合情境** | 市場對產品不熟悉、需要教育市場 |
| **首年營收預估** | NTD 1,500-3,000 萬（前 6 月低，後半年爬升）|

### 策略三：「既有客戶深耕」— 先吃自己的客戶，再向外擴

| 項目 | 內容 |
|------|------|
| **核心邏輯** | 從大豐既有 10,000+ B2B 客戶中篩選有地板需求的企業，交叉銷售，CAC 最低 |
| **優勢** | 獲客成本最低（已有關係）、信任基礎在、可搭配清運合約推銷 |
| **風險** | 既有客戶不見得有地板需求、「回收公司賣地板」的認知落差、可能量不大 |
| **適合情境** | 既有客戶中有足夠的地板更換需求 |
| **首年營收預估** | NTD 1,000-2,500 萬 |

## A6. 首選建議：策略一+策略二混合

**建議方案：「通路先行 + 案例驅動」雙軌並進**

| 月份 | 通路先行（量） | 案例驅動（質） |
|------|-------------|--------------|
| 4 月 | 拜訪 30 家潛在經銷商+建商+既有客戶 | 鎖定 3 個高能見度案場開始洽談 |
| 5 月 | 簽約首批 5-10 家經銷商 | 確認 1-2 個案場進入施工 |
| 6 月 | 經銷商首批叫貨 | 完成第 1 個案場，產出案例素材 |
| 7-9 月 | 擴大至 20 家經銷商+建材展曝光 | 完成 3 個案場，案例上線官網 |
| 10-12 月 | 靠案例+認證加持，經銷商主動推廣 | 用案例數據說服更多大型客戶 |

**選擇此方案的理由**：

1. **通路是量的保證**：經銷商/建商是 B2B 建材的主要流通管道，不走通路就沒量
2. **案例是質的保證**：沒有案例，經銷商也不敢主推。「你有哪些案場做過？」是 B2B 客戶的第一個問題
3. **兩者互相加速**：案例幫經銷商賣貨，經銷商幫擴大案例
4. **大豐的獨特資源剛好支撐兩軌**：清運車隊+拆除工班做案例的一條龍服務、B2B 客戶關係做經銷商開發

# Part B：行銷文案策略

## B1. 產品與受眾重述

### 產品本質

大豐環保獨家經銷的 PVC 循環再生地板：
- 目前已添加部分再生料製造
- 核心差異：回收後可 **100% 循環再製**，不產生廢棄物
- 提供「拆除→回收→鋪設」一條龍服務（競爭者做不到）
- 接單生產、零庫存、MOQ 240 坪/花色

### 改善文案的關鍵問題

| 問題 | 對文案的影響 |
|------|------------|
| 長虹產品的具體規格（厚度、耐磨等級、防火等級）？ | 決定能否打「品質不輸進口」的訴求 |
| 回收料佔比目前是多少%？ | 決定 GRS 認證等級+環保訴求的可信度 |
| 售價確定後與競品的價差是多少？ | 決定「省錢」還是「加值」的文案方向 |
| 有無已完成的施工案例照？ | 決定是否能用視覺素材支撐文案 |
| 長虹品牌在業界的口碑如何？ | 決定是否值得在文案中提及製造夥伴 |

## B2. 客戶洞察與核心訊息

### 理想買家畫像

**主要買家 #1：建材經銷商老闆**
