# 永續地板網站 UI/UX 競品分析報告

> 調查日期：2026-03-06
> 研究對象：Tarkett、Interface、Shaw Floors、Floori.io

---

## 一、Tarkett — 循環經濟頁面

**來源**：tarkett-group.com/en/sustainability/circular-economy/

### 1. 永續/循環經濟概念的視覺呈現

- **以真實攝影取代抽象圖表**：採用回收設施實景照（如荷蘭 Waalwijk 地毯回收中心），強調「可驗證的真實性」
- **未使用傳統環形迴圈圖**，而是透過廢料轉換過程的連續影像，以「紀實攝影」手法敘述循環故事
- 按地區（德國、荷蘭、加拿大、北美、歐洲）組織案例，展現全球實踐的廣度

### 2. Hero/Banner 設計

- **實景攝影**：寬幅橫向照片（1920x1281px），展示回收設施與廢料處理流程
- 風格偏向「工業紀實」而非「品牌形象」，突出製造端的真實場景

### 3. 產品展示方式

- 頁面以循環經濟故事為主軸，**非以產品花色為中心**
- 產品出現在回收案例的脈絡中（例如：回收地毯轉製新產品）
- 主站另有獨立的產品頁提供 room scene

### 4. 數據視覺化

- **粗體大數字 + 比喻轉換**：
  - 「124,000 tonnes」回收成果
  - 「29 countries」覆蓋範圍
  - 「12 座艾菲爾鐵塔」的重量比喻
  - 「38,500 棵樹生長 10 年」的碳抵銷換算
- 以**敘事文字包裹數字**，而非圖表或儀表板

### 5. 配色、排版、字體

| 元素 | 設定 |
|------|------|
| 主色 | 深藍 #003057 |
| 強調色 | 綠 #00b388、黃 #daaa00 |
| 背景 | 白底 + 深灰文字 #222222 |
| 字體 | Source Sans Pro、Roboto、Georgia |
| 標題層級 | H2: 46px / H3: 34px |

### 6. CTA 設計

- 多層次導引：影片連結（"watch the video"）、深度閱讀（"READ MORE HERE"）、專業平台跳轉
- CTA 按鈕為圓角設計，使用主色調填充
- **無直接購買 CTA**，導向知識學習與品牌認同

### 設計理念總結

> Tarkett 的策略是「用事實說話」— 以紀實影像、具體數據和比喻化表達建立循環經濟的可信度，刻意避免抽象化的綠色行銷。

## 二、Interface — 全球永續地板先驅

**來源**：interface.com

### 1. 永續/循環經濟概念的視覺呈現

- **永續作為主導航等級議題**：與產品線（Carpet Tile / LVT）並列為一級導航
- 設有 "Carbon Negative Innovation" 和 "Driven by Circularity" 專區
- 導航結構包含：Carbon Negative 產品分類、Recycling 專頁、Product Transparency、Certifications & Documentation
- 提供互動式 **碳影響計算器**（"Calculate Your Carbon Impact"）連結

### 2. Hero/Banner 設計

- **產品情境照**：主 banner 為 IronGrass Herringbone 產品在空間中的實景（1920x900px）
- 以產品美學而非標語作為視覺主體
- 風格簡潔、讓產品圖像自己說話

### 3. 產品展示方式

- 支援 **room scene（情境空間照）** 和安裝實景照
- 花色展示支持多色彩變體（colorCount 系統）
- 產品分為標準線和 nora 橡膠地板兩類
- 購物流程分為 **Samples Cart（樣品車）** 和 **Materials Cart（材料車）**

### 4. 數據視覺化

- 碳計算工具為獨立互動頁面，非靜態數字展示
- 首頁本身未直接呈現大量統計數據
- 以導航引導使用者進入深度數據頁面

### 5. 配色、排版、字體

| 元素 | 設定 |
|------|------|
| 主題綠 | #50CC55（生態綠） |
| 深色調 | #2E4A21（深森林綠） |
| 暖色 | #a47864（土棕色） |
| 文字色 | #222222 |
| 字體 | Serif + Sans-serif 混合策略 |
| 風格 | 現代簡約，自然色系 |

### 6. CTA 設計

- 雙購物車系統：樣品 + 材料分流
- 帳戶系統完整（登入/登出）
- 全球 13+ 地區選擇器
- **碳計算器作為轉換漏斗入口**：引導使用者從「了解碳影響」到「選擇碳負排產品」

### 設計理念總結

> Interface 將永續性融入品牌 DNA，永續不是獨立的企業責任頁面，而是與產品選擇流程深度整合。碳計算器是核心差異化工具。

## 三、Shaw Floors — 美國主流地板品牌

**來源**：shawfloors.com

### 1. 永續/循環經濟概念的視覺呈現

- **首頁幾乎不提永續**：永續內容藏在 About > Sustainability 子頁面中
- 永續頁面採用 **"People + Planet" 敘事框架**，以情感化攝影（兒童與寵物在地板上互動）建立連結
- 使用深藍背景 + 白色文字營造專業權威感

### 2. Hero/Banner 設計

**首頁**：
- **影片 Banner**（MP4 格式）：家庭生活場景，柔軟地板上的溫馨畫面
- 標題：**"Make Every Step Feel More Like Home"**
- 副標強調情感連結：「盛大的慶祝、日常的歡樂」
- **雙 CTA**：「Browse Products」（深藍色填充）+「Why Shaw Floors」（白色邊框）

**永續頁面**：
- 傳統文字型 Hero，Navy 背景 + "People + Planet" 大標題

### 3. 產品展示方式

- 大量使用 **room scene 情境照**：客廳、廚房、浴室的生活場景
- 六大品類分區展示：Luxury Vinyl、Hardwood、Carpet、Laminate、Tile、Area Rug
- **Pet Perfect 產品線**特別突出：展示寵物與地板互動場景
- 圖片品質高，以居家生活方式為導向

### 4. 數據視覺化

**永續頁面**：
- 以**敘事性統計**取代圖表：「近 90% 產品獲 Cradle to Cradle 認證」
- 五大屬性 icon 網格：Material、Moisture、Sound、Air、Clean
- **三欄式認證展示**：Cradle to Cradle、Green Label Plus、GREENGUARD
- 無傳統數據儀表板

**首頁**：
- 完全沒有環保數據展示

### 5. 配色、排版、字體

| 元素 | 設定 |
|------|------|
| 主色 | 深藍色（Navy） |
| 輔色 | 松綠色（Teal） |
| 排版 | 多層級標題（xl / l / m）|
| 字體 | 現代 Sans-serif |
| 留白 | 寬敞、分層式內容佈局 |

### 6. CTA 設計

- 首頁三層漏斗：Hero 雙 CTA → 品類卡片 → 「Free Design Services」和「Financing」
- 永續頁面僅一個 CTA：「See All Sustainability Reports」（次要樣式 + 右箭頭 icon）
- 整體轉換導向**購買流程**而非永續認知

### 設計理念總結

> Shaw Floors 以「生活方式品牌」定位，永續是加分項但非核心賣點。視覺策略聚焦於溫暖的家庭情境，永續認證以「可信度背書」方式低調呈現。

## 四、Floori.io — 地板 AR 視覺化技術平台

**來源**：floori.io

### 1. 產品/服務視覺呈現

- **科技感設計**：大量 SVG 圖示、漸層色彩
- 豐富的 CSS 動畫（fadeIn、zoomIn、slideInRight）
- 採用「數位雙胞胎」視覺語言
- 四象限生態系統架構圖：DIGITIZE → INNOVATE → OPTIMIZE → GROW

### 2. Hero/Banner 設計

- **混合媒體**：實景照與渲染圖的動態輪播
- 標題：「AI Tools to Visualize, Specify & Sell Floors, Tiles, Walls & More」
- 功能導向的標語，而非情感訴求
- 雙 CTA：「Contact Sales」+「Try Yourself」（Demo 優先）

### 3. AR 視覺化功能展示

- **六種表面類型輪播**：地板、天花板、油漆、牆面、櫥櫃、窗戶
- 滑鼠懸停暫停 + 手動選擇切換
- **Before/After 比較滑桿**：隨滾動觸發，展示產品前後對比
- 互動式 pin 導航，可在 room scene 中點選不同區域
- 強調「客戶上傳自己的房間照片」功能

### 4. 產品展示方式

- 以**應用情境**為主，非色票或材質特寫
- 品牌客戶案例影像（Swiss Krono、Barlinek、Rug Love）
- 技術規格展示：BIM 模型、UV 貼圖、PBR 映射
- 功能卡片網格佈局

### 5. 配色、排版、字體

| 元素 | 設定 |
|------|------|
| 主色 | 深紫 #5636d1 |
| 強調色 | 橘紅 #F15A22、深粉紅 #9E2062 |
| 字體 | Montserrat（標題）、Roboto（內文） |
| 排版 | 左對齊標題、居中描述、卡片網格 |
| 背景 | 漸層 + 實景照 + 純白分段交替 |

### 6. CTA 設計

- **多觸點轉換設計**：
  - 「Book a demo」倒數計時器（強調 55 秒內回電）
  - 多步驟表單：地區 → 語言 → 聯繫資訊 → 業務類型
  - HubSpot 整合的會議預約系統
  - 「Call Me」回撥小工具
- 定位 30 億美元地板產業的解決方案

### 7. 永續相關內容

- **無明確永續訴求**
- 間接價值：數位化減少實體樣品製作 → 減少材料浪費
- 聚焦於銷售效率與客戶體驗

### 設計理念總結

> Floori 是純技術 B2B 平台，以 AR/AI 視覺化為核心賣點。雖無直接永續訴求，但其「數位化取代實體樣品」的模式本身具備減廢價值。

## 五、永續地板網頁最佳實踐摘要

### A. 永續概念的視覺呈現策略

| 策略 | 代表品牌 | 適用場景 |
|------|---------|---------|
| **紀實攝影**（回收設施、生產流程實拍） | Tarkett | 強調製造端的循環實踐、建立可信度 |
| **融入產品導航**（永續作為一級選單） | Interface | 永續是品牌核心 DNA |
| **認證背書式**（低調展示環保認證） | Shaw Floors | 永續是加分項而非主打 |
| **數位化減廢**（間接永續價值） | Floori | 技術驅動的間接環保貢獻 |

### B. 數據視覺化最佳做法

1. **大數字 + 比喻轉換**（Tarkett）：「124,000 噸 = 12 座艾菲爾鐵塔」— 讓抽象數據具象化
2. **互動式計算器**（Interface）：碳影響計算器讓使用者「算出自己的貢獻」— 提升參與感
3. **敘事性統計**（Shaw）：「近 90% 產品獲認證」— 簡潔有力的單一數據點
4. **認證 icon 網格**（Shaw）：五大屬性圖示 — 快速掃描、降低認知負擔

### C. 配色趨勢

| 色系 | 用途 | 品牌範例 |
|------|------|---------|
| 深藍 / Navy | 信任、專業權威 | Tarkett #003057、Shaw Navy |
| 生態綠 | 永續主題標記 | Interface #50CC55、Tarkett #00b388 |
| 土棕 / 暖色 | 自然材質感 | Interface #a47864 |
| 白底 + 深灰文字 | 乾淨閱讀體驗 | 所有品牌共通 |

### D. Hero Banner 設計模式

| 類型 | 效果 | 品牌 |
|------|------|------|
| 影片 Banner | 最高沉浸感、展示生活方式 | Shaw Floors |
| 產品情境照 | 產品美學為主，簡潔專業 | Interface |
| 紀實工業攝影 | 強調製程真實性 | Tarkett |
| 動態輪播 + 渲染圖 | 科技感、功能展示 | Floori |

### E. 產品展示最佳實踐

1. **Room Scene 是標配**：所有品牌都使用情境空間照（客廳、廚房、商辦）
2. **多色彩變體切換**：Interface 提供同產品不同花色的即時切換
3. **AR 預覽是進階差異化**：Floori 的「上傳自己房間 + 即時預覽」代表未來趨勢
4. **Before/After 比較滑桿**：展示安裝前後效果的直覺化工具

### F. CTA 設計策略

| 層級 | 設計建議 | 範例 |
|------|---------|------|
| 認知層 | 「了解我們的循環經濟」影片/故事 | Tarkett: "watch the video" |
| 參與層 | 碳計算器、AR 體驗 | Interface: "Calculate Your Carbon Impact" |
| 轉換層 | 索取樣品、聯繫業務 | Interface: Samples Cart / Shaw: Free Design Services |
| 急迫感 | 倒數計時、即時回撥 | Floori: 55 秒回電承諾 |

### G. 關鍵洞察與建議

1. **永續故事要分層敘述**：Tarkett 模式（大數字入口 → 地區案例 → 客戶見證）是最完整的永續敘事結構
2. **互動工具比靜態數據更有效**：Interface 的碳計算器將「被動閱讀」轉為「主動參與」
3. **認證要視覺化但不要喧賓奪主**：Shaw 的三欄認證展示是好的平衡點
4. **永續頁面不應孤立存在**：Interface 將永續融入產品導航是最理想的資訊架構
5. **比喻化數據是最佳記憶錨點**：「12 座艾菲爾鐵塔」比「124,000 噸」更容易被記住和分享
6. **AR/數位體驗是未來趨勢**：Floori 的 Before/After 滑桿和 Room Scene 互動可作為產品頁的差異化功能
